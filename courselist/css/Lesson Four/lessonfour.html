Lesson 4: Typography and Fonts

What is Typography?

  • Typography is the art and technique of arranging text. It involves selecting typefaces, adjusting sizes, spacing, and line lengths to ensure that text is readable, visually appealing, and effective.

  • Typography plays a crucial role in web design, as it helps convey a message clearly while making the content look attractive and professional.

Basic Typography Concepts

  • Fonts and Typefaces: A font is a set of characters in a specific style, size, and weight. A typeface refers to the overall design of the set of fonts.

  • Font Size: The size of the text is measured in points (pt). Typical body text is between 12px and 16px, while headings are larger to emphasize importance.

  • Font Weight: Font weight controls the thickness of the text. Common values include normal and bold, but there are many options like lighter, bolder, etc.

  • Line Height: Line height defines the space between lines of text. Itâ€™s essential for readability, particularly in larger blocks of text.

How to Use Fonts in CSS

  • To use fonts in CSS, you can set the font-family property to choose the desired font. Here's an example of how to use the Arial font: 

    font-family: Arial, sans-serif;

  • You can also use web fonts from services like Google Fonts by adding a link tag in your HTML file: 

          <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
    Then, apply the font in your CSS: 
    font-family: 'Roboto', sans-serif;

Practical Exercise

  • Step 1: Choose a font family for your webpage from Google Fonts.

  • Step 2: Add the chosen font to your HTML file and apply it to the body of your page.

  • Step 3: Experiment with font sizes, line heights, and font weights to make your text more readable and visually appealing.

  • Step 4: Adjust the spacing between paragraphs to improve the layout of your content.
