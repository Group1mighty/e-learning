Lesson 4: Working with Images

Adding Images in HTML

  • The `` tag is used to embed images in HTML. It is a self-closing tag and requires the following attributes:

    • src: Specifies the path to the image file (e.g., src="image.jpg").
    • alt: Provides alternative text for the image, improving accessibility and SEO.

  • Example: 

    <img src="example.jpg" alt="Example Image">

Resizing and Styling Images

  • You can adjust the size of an image using the following:

    • HTML attributes: Use width and height attributes directly inside the `` tag.
    • CSS: Use the width and height properties in your stylesheet for more control.

  • Example with CSS: 

    
<img class="responsive" src="image.jpg" alt="Responsive Image">
<style>
  .responsive {
    width: 100%;
    height: auto;
  }
</style>
