Lesson 1: Introduction to CSS

Introduction to CSS

  • CSS (Cascading Style Sheets) is a language used to describe the presentation of a web page written in HTML or XML. It controls the layout, colors, fonts, and overall appearance of a webpage.

  • The main purpose of CSS is to separate content (HTML) from design (CSS). This helps with maintenance, as changes in design can be applied universally without affecting the structure of the web page.

  • How does CSS work with HTML?

    • CSS rules are made up of selectors and declarations.
    • Selectors target HTML elements, and declarations specify how the elements should be styled.
    • For example, p { color: blue; } changes the text color of paragraph elements to blue.

  • Ways to apply CSS to HTML:

    • Inline CSS: Adding CSS directly to HTML elements using the style attribute. like <p style="color: red;">.
    • Internal CSS: Placing CSS within the <style> tag inside the HTML document's <head> section. like <style> p { color: red; } </style>.
    • External CSS: Linking to an external stylesheet using the <link> tag in the <head> section. like <link rel="stylesheet" href="styles.css">.

  • CSS Syntax:

    • Selector: The HTML element you want to style.
    • Property: The aspect of the element you want to change (e.g., color, font-size, margin).
    • Value: The setting for the property (e.g., red, 12px, 10px).

Practical Exercise

  • Step 1: Create a simple HTML file with a heading and paragraph.

  • Step 2: Apply CSS styles to the elements using inline, internal, or external methods.

  • Step 3: Experiment with different properties like color, background-color, font-size, and margin to see how they affect the webpage.
