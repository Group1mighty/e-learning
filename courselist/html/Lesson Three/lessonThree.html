Lesson 3: Adding Text and Formatting

Lesson Overview

  • Adding Headings:

    HTML provides six levels of headings, from <h1> (largest) to <h6> (smallest). Use these to structure your content.

    <h1>This is a main heading</h1>
  • Paragraphs:

    To add paragraphs, use the <p> tag. Paragraphs create spacing around text to improve readability.

    <p>This is a paragraph of text.</p>
  • Text Formatting:
    • <b>: Bold text. Example: <b>Bold Text</b>
    • <i>: Italicize text. Example: <i>Italic Text</i>
    • <u>: Underline text. Example: <u>Underlined Text</u>
  • Lists:

    HTML offers two types of lists:

    1. Ordered List: Use <ol> with <li> for numbered items.
    2. Unordered List: Use <ul> with <li> for bullet points.
    3. Defination list: Use <dl> with <dt> for bullet points. and with <dd> for defination
    
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>

<dl>
  <dt>HTML</dt>
  <dd> - Hypertext Markup Language. <dd>
  <dt>CSS</dt>
  <dd> - cascading style sheet. <dd>
  <dt>JS</dt>
  <dd> - JavaScript <dd>
</dl>
