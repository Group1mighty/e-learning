Lesson 1: Introduction to HTML

Introduction to HTML

  • HTML (HyperText Markup Language) is the standard language for creating web pages and web applications.

  • HTML provides the basic structure of a web page, which is enhanced and modified by technologies like CSS and JavaScript.

  • Key Components of HTML:

    • Elements: These define the structure and content of a webpage, such as paragraphs, headings, images, and links.
    • Tags: Tags are used to mark the beginning and end of an element, like <p> for paragraphs.
    • Attributes: Attributes provide additional information about an element, such as class or id.

  • Why Learn HTML?

    1. It is the foundation of web development.
    2. Understanding HTML helps in building and structuring websites.
    3. It enables better collaboration with designers and developers.

Practical Exercise

  • Step 1: Create a new HTML file with the extension .html.

  • Step 2: Write a basic HTML structure using the <html>, <head>, and <body> tags.

  • Step 3: Add content such as headings, paragraphs, and links.
