Lesson 2: Selectors and Properties

Understanding CSS Selectors and Properties

  • What Are CSS Selectors?
    CSS selectors are used to target HTML elements so that you can apply styles to them. There are various types of selectors you can use:

    • Element Selector: Targets all elements of a specific type (e.g., p for all paragraphs).
    • Class Selector: Targets elements with a specific class attribute (e.g., .class-name for all elements with the class class-name).
    • ID Selector: Targets an element with a specific ID attribute (e.g., #id-name for the element with ID id-name).
    • Universal Selector: Targets all elements on the page (e.g., *).
    • Attribute Selector: Targets elements with a specific attribute (e.g., [type="text"] for all input elements with type text).

  • Understanding CSS Properties:
    CSS properties define how a specific style is applied to an element. Some common CSS properties include:

    • color: Sets the text color of an element.
    • font-size: Specifies the size of the text.
    • background-color: Defines the background color of an element.
    • margin: Controls the space outside the element.
    • padding: Controls the space inside the element.
    • border: Adds a border around the element.

    Practical Exercise:
    Let's apply CSS selectors and properties:

    1. Apply Styling to Paragraphs

    1. Target all paragraphs and change the font size to 18px:
      • p { font-size: 18px; }

    2. Style Elements with a Class Selector

    1. Apply a background color to all elements with the class highlight:
      • .highlight { background-color: yellow; }

    3. Use ID Selector to Change Text Color

    1. Change the text color of an element with ID main-heading:
      • #main-heading { color: blue; }

    Summary:

    • CSS Selectors: Element, Class, ID, Universal, Attribute
    • CSS Properties: color, font-size, background-color, margin, padding, border
    • Practical Skills:
      • Apply styles using different CSS selectors.
      • Use various CSS properties to style elements effectively.
