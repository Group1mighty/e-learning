Lesson 5: Responsive Design Basics

What is Responsive Design?

  • Responsive web design is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. The core of responsive design is the ability to adjust the layout and content dynamically to suit the device viewing the site, ensuring optimal user experience across smartphones, tablets, and desktops.

Flexible Layouts

  • Flexible Layouts: This method uses relative units, such as percentages, rather than fixed units like pixels. This enables elements to resize according to the screenâ€™s width.

    For example, setting a container to 80% of the screen width allows the content to adjust accordingly on various screen sizes.

      .container {
    width: 80%;
  }

Media Queries

  • Media Queries: Media queries enable the application of specific styles based on the device's characteristics, such as width, height, or orientation. They allow for different CSS rules to be applied to different screen sizes.

      @media only screen and (max-width: 600px) {
    .container {
      width: 100%;
    }
  }

    In this example, the container's width is set to 100% for devices with a screen width of 600px or smaller, ensuring a full-width layout on mobile devices.

Mobile-First Design

  • Mobile-First Approach: A mobile-first strategy means designing for the smallest screen sizes first and then adding more complex layouts for larger screens. This method prioritizes performance and usability for mobile users, who represent the largest audience for web browsing today.

    By starting with the smallest screens and gradually enhancing the design for larger screens using media queries, developers ensure a seamless experience across all devices.

Practical Exercise

  • Step 1: Create a flexible layout for a basic webpage. Start by setting the container's width to 80% using percentage-based values. 

      .container {
    width: 80%;
  }

  • Step 2: Implement media queries to change the layout when the screen width is below 600px. 

      @media only screen and (max-width: 600px) {
    .container {
      width: 100%;
    }
  }
