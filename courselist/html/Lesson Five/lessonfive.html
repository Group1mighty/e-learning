Lesson 5: Creating Links

Understanding Hyperlinks

  • Hyperlinks, or links, allow users to navigate between web pages or sections of a page. They are essential for creating an interconnected and user-friendly web experience. There are three main types of links:

    • External Links: Direct users to other websites by using absolute URLs (e.g., https://example.com).
    • Internal Links: Navigate to other pages within the same website using relative paths (e.g., about.html).
    • Anchor Links: Jump to specific sections on the same page by referencing an elementâ€™s ID (e.g., #section-id).
    Using these types of links appropriately ensures smooth navigation and enhances user experience.

How to Create Links

  • Basic Syntax: Links are created using the <a> (anchor) tag in HTML. The href attribute specifies the destination URL.

      <a href="https://example.com">Visit Example</a>

    Example: Clicking the link above will open the "Example" website in the browser.

  • Open in New Tab: Use the target="_blank" attribute to open a link in a new tab.

      <a href="https://example.com" target="_blank">Open Example in a New Tab</a>

    This is useful for external links, ensuring users remain on your site while exploring other resources.

  • Anchor Links: Link to specific sections within the same page by using IDs and the # symbol.

      <a href="#contact-section">Go to Contact</a>
  ...
  <div id="contact-section">Contact Us</div>

    This allows users to jump to the "Contact Us" section of the page.

Practical Exercise

  • Step 1: Create an external link to a favorite website. 

      <a href="https://www.wikipedia.org" target="_blank">Visit Wikipedia</a>

    This link opens the Wikipedia homepage in a new tab.

  • Step 2: Create an internal link to another page on your website. 

      <a href="about.html">Learn More About Us</a>

    Ensure the file name matches the destination page on your server or local project folder.

  • Step 3: Add an anchor link to a specific section within a page. 

      <a href="#services">See Our Services</a>
  ...
  <section id="services">Our Services</section>

    Clicking this link scrolls the page directly to the "Services" section.
